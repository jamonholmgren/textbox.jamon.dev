# Textbox

This is a simple textbox for writin quick notes that stick around between sessions.

## Features

- Simple and easy to use
- No login required
- No tracking
- No ads
- No cookies
- No data collection

## How to use

1. Go to [textbox.jamon.dev](https://textbox.jamon.dev)
2. Start typing
3. Your notes will be saved and you can come back to them later

## How to contribute

1. Fork the repository (https://github.com/jamonholmgren/textbox.jamon.dev)
2. Make your changes
